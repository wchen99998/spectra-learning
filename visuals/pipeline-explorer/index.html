2. Valid Mask & Padding

How many of the 60 peak slots are filled vs zero-padded across a batch of 16 spectra. The valid/padding layout is invariant across all augmented views.

3. Multi-Crop Views

Each spectrum is augmented into 6 views: 2 global (keep 100%, jitter only) + 4 local (keep 25%, mask ~75% of valid peaks). Global views see all valid peaks; local views get aggressive masking for the self-supervised objective. fused_masked_positions tracks which peaks are masked per view.

5. Jitter Effect

Each augmented view applies Gaussian jitter to m/z (σ=0.0001) and intensity (σ=0.001). The point clouds below show 30 augmentations of the same spectrum's first 15 valid peaks (global view 1). Red markers show original values.

7. Batch Statistics

Aggregate statistics across fused tensors: m/z and intensity distributions by peak state (kept / masked / padded), and masking rates for global vs local views.

Data: 48 real mass spectra from GeMS-A training set (3 batches × 16), each with 6 augmented views (2 global + 4 local). 30 jitter augmentation runs captured for one spectrum. Extracted via extract_data.py.
